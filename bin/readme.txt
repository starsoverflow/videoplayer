
命令行使用：
                       videoplayer	[/nooverwrite] [/nosavehistory] [/mediafile="Your media file path"]
                       [/playlist="Your playlist path"] [/keepwidth=800] [/size=800,450] [/add]

                       /nooverwrite	退出时不保存播放列表
                       /nosavehistory	退出时不保存历史记录
                       /playlist	指定播放列表文件
                       /mediafile	指定要播放的媒体文件

                       以下选项只有指定了/mediafile才有效:
                       /keepwidth	使窗口保持指定的宽度和视频的纵横比
                       /size		指定窗口大小
                       /add		将媒体文件添加到已经在运行的播放器实例

                       如果同时指定了/playlist和/mediafile，只有/mediafile生效
